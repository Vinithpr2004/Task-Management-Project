🚀 Full Stack Task Management System (Rust + WebAssembly + JS):

A high-performance full-stack task management application built using Rust compiled to WebAssembly (WASM) and integrated with a JavaScript frontend. The system leverages Rust for both client-side and backend logic, ensuring speed, memory safety, and efficient execution.

The frontend is developed using modern JavaScript frameworks and communicates with Rust-powered WebAssembly modules for performance-critical operations. The backend is implemented in Rust, exposing secure APIs and handling business logic efficiently. Data is stored and managed using PostgreSQL, ensuring reliability and scalability.

🔑 Key Features:
⚡ Rust + WebAssembly Integration for high-performance computation
🔐 JWT-based Authentication and role-based access control
📦 RESTful APIs built with Rust backend
✅ Task creation, updating, filtering, and status tracking
🔄 Real-time UI updates with seamless WASM-JS interaction
🛡️ Memory-safe and efficient system design using Rust
📱 Responsive and user-friendly interface

🛠️ Tech Stack:
Frontend: JavaScript (React / Vanilla JS)
WASM Layer: Rust → WebAssembly
Backend: Rust (Actix / Axum)
Database: PostgreSQL
Auth: JWT

🎯 Highlights:
This project demonstrates the power of combining Rust + WebAssembly with modern web technologies to build scalable, secure, and high-performance applications beyond traditional JavaScript-based stacks.
